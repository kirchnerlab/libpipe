Libpipe Documentation

Introduction

Libpipe is a lightweight C++ pipelining library.

Installation

Use cmake and make. For more details see Installation.

Usage

see Usage.

Examples

libpipe is provided with 4 examples.

example-strings-mtime
shows the modification time manager with compile time configurations
example-suupercase-strings
shows the compile time configuration of libpipe
example-rtc
shows the runtime configuration implementation of libpipe
example-matrices
shows the runtime configuration implementation of libpipe, and shows the speedup of the threaded implementation of libpipe
